Selamat Datang

Ini adalah website sederhana yang dibuat menggunakan bahasa HTML. Website ini dibuat sebagai tugas ujian praktek siswa SMK jurusan Teknik Komputer dan Jaringan.Berikut adalah poin-poin penting mengenai pembelajaran HTML bagi siswa :

  1. Dasar dan fondasi Web : html adalah tulang punggung setiap situs web yang ditampilkan oleh browser.Memahami HTML memberikan pemahaman tentang cara kerja internet dan pengembangan situs web.
  2. pembuatan Konten: HTML memungkinkan siswa membuat elemen konten, termasuk teks,gambar dan tautan (hyperlink)
  3. Mudah Dipelajari: Html relatif mudh dipelajari bahkan bagi pemula,menjadikannya titik awal yang ideal dalam dunia Web
